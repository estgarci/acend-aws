# Acend

## Description
Acend is a full-stack (MERN) airline website that communicates with a proprietary API and displays the real-time list of flights of a selected airport.

The purpose of this project was to build a website that would showcase my abilities as a full-stack software engineer. The goal was to create a production-ready application and focus on the search-flights-widget. 

## Features:

### The Front
* Designed using AdobeXD
* Built with React & Bootstrap
* State managed with Redux
* Handled CSS using SCSS and Styled-Components
* Implemented AirBnB's date-picker library

### The Middle
* Deployed using an AWS EC2 instance that runs a linux server with ubuntu20.4
* Configured an Nginx reverse proxy server
* Configured a secured communication using SSL 

### The Back
* Customized a backend server with multiple endpoints
* Configured proxy server for communication with FlightAware, a proprietary API
* Implemented user authentication through github and facebook using OAuth2.0
* Handled user sessions with JWT tokens
* Configured backend server to allow cross-origin requests
